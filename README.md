# Краткое описание
Условно бесплатная CRM систсема.
Выполняет базовый функционал, который более чем достаточен для небольшой компании.
Для расширения возможностей необходимо докупать плагины.


# Как пользоваться

переиименуйте

## Установка
```
docker-compose up -d
```

## Удаление
```
docker-compose down
```
